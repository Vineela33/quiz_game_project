# 🎮 Quiz Game (Mini Project)

A simple and interactive **Quiz Game** built to test knowledge through multiple-choice questions. This project is beginner-friendly and demonstrates core programming concepts like loops, conditionals, and user input handling.

---

## 📌 Features

* Questions will be displayed
* Score tracking system
* Instant feedback on answers
* Clean and simple command-line interface
* Easy to customize questions

---

## 🎯 How to Play

1. Run the program
2. Read each question carefully
3. Enter the correct answers.
4. Get your final score at the end
